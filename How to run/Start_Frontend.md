# How to Start the Frontend Server

The frontend of the Writing Tools Web App is built with Vite and React. It needs to run simultaneously with the backend server.

1. **Open a new terminal window** (Command Prompt or PowerShell).
   *(Make sure this is a separate window from the one running your backend).*

2. **Navigate to the frontend directory:**

   ```powershell
   cd c:\Users\Paradox-Labs\Documents\Projects\Writing_Tools\frontend
   ```

3. **Start the development server:**

   ```powershell
   npm run dev
   ```

4. **Open the App:**
   Once the server starts, it will output a local URL (usually `http://localhost:5173`).
   Hold `Ctrl` and click the link in your terminal, or open your web browser and navigate to that URL to use the application!
